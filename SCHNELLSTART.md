# 🚀 MASESites AG - Website Schnellstart

## ✅ SOFORT STARTEN (Ohne Installation):

### Methode 1: Python Server (Empfohlen)
```bash
python start-server.py
```

Dann öffne im Browser: **http://localhost:3000**

### Methode 2: Direkt im Browser
Öffne einfach die `index.html` im Browser (Doppelklick).

**⚠️ Wichtig:** Kontaktformular funktioniert nur mit Server!

---

## 📧 Kontaktformular aktivieren (Optional):

### Voraussetzungen:
- Node.js installiert
- Gmail App-Password

### Setup:
1. **Dependencies installieren:**
   ```bash
   npm install
   ```

2. **.env erstellen:**
   - Kopiere `.env.example` zu `.env`
   - Trage Gmail App-Password ein

3. **Server starten:**
   ```bash
   npm start
   ```

Website: **http://localhost:3000**

## 🔧 Troubleshooting:

### Website ist leer?
- ✅ Verwende `python start-server.py` statt Datei direkt zu öffnen
- ✅ Oder öffne über `http://localhost:3000`

### Kontaktformular funktioniert nicht?
- ✅ Server muss laufen (`npm start` oder Python-Server)
- ✅ `.env` Datei mit Gmail-Credentials erstellen

### Navigation funktioniert nicht?
- ✅ Server verwenden (nicht file://)
- ✅ Alle HTML-Dateien müssen im selben Ordner sein

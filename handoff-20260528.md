## 1. NAVBAR — Frosted Glass (Apple / Linear / Stripe Style)

**Problem:** Header hatte zwei separate dunkle Hintergrundboxen (Header-Bar + Logo-Box), die übereinander lagen und wie ein "Viereck im Viereck" aussahen.

**Root Cause:** `--header-bg: #0d1117` (solid opak) + Logo-PNG mit eigenem dunklen Hintergrund → zwei verschiedene Solid-Farben gestapelt.

**Fixes in `styles.css`:**
```css
/* CSS-Variable auf rgba umgestellt */
--header-bg: rgba(255, 255, 255, 0.78);  /* light */
--header-bg: rgba(10, 11, 18, 0.76);     /* dark */

/* Frosted Glass */
.site-header {
  background: var(--header-bg);
  backdrop-filter: saturate(180%) blur(20px);
  -webkit-backdrop-filter: saturate(180%) blur(20px);
  border-bottom: 1px solid rgba(255,255,255,0.055);
}
```

**Logo-Box-Fix:**
```css
/* mix-blend-mode: screen lässt dunkle Pixel des PNG verschwinden */
.dark .logo-img {
  mix-blend-mode: screen;
  filter: brightness(1.12) contrast(1.05);
}
```

**Nav-Links (alle in derselben Alpha-Familie):**
- Inactive: `rgba(255,255,255,0.50)`
- Hover: `rgba(255,255,255,0.88)` + `rgba(255,255,255,0.06)` Hintergrund
- Active: `rgba(255,255,255,0.95)` + `rgba(255,255,255,0.08)` Hintergrund
- CTA-Button: Ghost-Stil `rgba(255,255,255,0.09)` + `border: 1px solid rgba(255,255,255,0.18)`

## 2. LOGO-LINK BUG — Startseite nicht erreichbar

**Problem:** Klick auf Logo führte nicht zur Startseite — `href` wurde zu `?lang=de` statt `/?lang=de`.

**Root Cause:** `theme.js` → `updateInternalLanguageLinks()` entfernt führenden `/` von jedem Pfad (`path.slice(1)`). Bei `href="/"` wurde der Pfad leer → `href="?lang=de"` (relative URL = gleiche Seite).

**Fix in `theme.js` (Zeile ~1470):**
```javascript
// Vorher:
if (path.indexOf('/') === 0) path = path.slice(1);
a.setAttribute('href', path + url.search + url.hash);

// Nachher:
if (path.indexOf('/') === 0) path = path.slice(1);
if (!path) path = '/';  // Root "/" case — don't lose the slash
a.setAttribute('href', path + url.search + url.hash);
```

## 5. ANIMATIONS — Emil Kowalski Design Engineering

**Philosophie:** Taste is trained. Unseen details compound. Interactions must feel responsive.

### Fixes in `mase-animations.css`:

| Element | Vorher | Nachher |
|---------|--------|---------|
| Word-reveal | `blur(8px)`, `720ms` | `blur(5px)`, `620ms` |
| Badge-Eintritt | `scale(0.9)`, `560ms` | `scale(0.95)`, `420ms` |
| Badge-Delays | `550/650/750ms` | `480/560/640ms` |
| Scroll-reveal | `translateY(28px)`, `700ms` | `translateY(20px)`, `560ms` |
| Stagger-Delays (Trust Cards) | `60/160/260/360ms` | `0/60/120/180ms` |
| Stagger-Delays (Process) | `80/200/320/440/560ms` | `0/55/110/165/220ms` |
| Shimmer-Sweep | `700ms` | `420ms` |
| Sticky-CTA Eintritt | `scale(0.92)`, `380ms` | `scale(0.96)`, `280ms` |

### Neuer Block in `styles.css` — Interaction Polish:

```css
:root {
  --ease-out-strong: cubic-bezier(0.23, 1, 0.32, 1);
}

/* Button press feedback */
.button:active { transform: scale(0.97); }
.back-to-top:active { transform: scale(0.92); }
.dark-toggle:active { transform: scale(0.91); }
.nav-toggle:active { transform: scale(0.93); }

/* Hover nur auf Pointer-Geräten */
@media (hover: hover) and (pointer: fine) {
  .trust-card:hover { transform: translateY(-4px); }
  .process-step:hover { transform: translateY(-3px); }
  .site-footer a:hover { transform: translateX(3px); }
}

/* Hero-Stats Stagger-Einblendung */
.hero-stats .hero-stat { animation: mase-statIn 480ms forwards; }
/* Delays: 800/860/920/980/1040ms */

/* Theme-Switch-Transition beschleunigt */
/* 0.45s → 0.22s */
```

## 6. EINHEITLICHE SEITENHINTERGRÜNDE

**Problem:** Jede Unterseite hatte einen anderen Hintergrund:
- `leistungen-hero`: Light-Mode-Blaugradient (`#f0f9ff → #f6f9ff`)
- `ki-hero`: Light-Mode-Grüngradient + eigener Dark-Mode-Gradient
- `ktk-hero` (Kontakt): Hardcoded Navy-Gradient (`#070c1f`)
- `ueber-uns.html` CTA: Inline `background: #f8fafc` (bricht dark mode)

**Fix in `styles.css` (neuer Block am Ende):**
```css
/* Alle Unterseiten-Heroes → gleiche Basis */
.page-hero, .leistungen-hero, .ki-hero, .ktk-hero {
  background: var(--bg) !important;
  border-bottom: 1px solid var(--line);
}

/* CTA-Sections → einheitlich */
.cta-section {
  background: var(--surface) !important;
  border-top: 1px solid var(--line);
}
```

**Fix in `ueber-uns.html`:**
```html
<!-- Vorher: -->
<section class="cta-section" style="background: #f8fafc; padding: 4rem 0;">
<!-- Nachher: -->
<section class="cta-section" style="padding: 4rem 0;">
```

**Resultat:** Alle Unterseiten nutzen `var(--bg)` als Basis — theme-aware, kein Chaos.

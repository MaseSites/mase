# Clean Inventory Web App

## Start

1. `npm install`
2. `npm run db:push`
3. `npm run dev`

## Features

- White, calm business UI
- Read-only by default
- Admin edit mode with password `admin`
- One-click stock actions for inbound and sales
- SQLite persistence with product, movement, and sales records
- Uploadable local product images
- Chart-based analytics with day, week, month, and year filters
